Model-level and surface-level data was downloaded with scripts
`download_plevels.py` and `download_surface.py` respectively using the ECMWF
data API. Do use this download method you will have to register for an ECMWF
account and use your own key to access their API. Details are on:

- https://software.ecmwf.int/wiki/display/WEBAPI/Python+ERA-interim+examples?src=contextnavpagetreemode

- API key: https://api.ecmwf.int/v1/key/
